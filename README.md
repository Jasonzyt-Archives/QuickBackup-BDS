# QuickBackup-BDS
A QuickBackup Plugin Of Bedrock Dedicated Server

### Plugin Commands
#### Server Commands
+ qb_backup   
  Run once backup (NOT AutoBackup)
+ qb_reload   
  Reload config file
+ qb_fix  
  Run fix function:  
  Downlaod config file and bandzip console application
